# Student Math Score Prediction 🎯

An End-to-End Machine Learning project to predict a student's Maths score based on different academic and personal attributes.

#### This project includes:

Data preprocessing and model training in Jupyter Notebook
Multiple Machine Learning algorithms comparison
Complete ML pipeline implementation
Flask-based web application for prediction
Simple frontend interface for user input

#### 📌 Project Overview

The goal of this project is to predict the math score of students using Machine Learning techniques.

#### The project follows a complete ML workflow starting from:

Data collection
Data preprocessing
Exploratory Data Analysis (EDA)
Model training and evaluation
Model selection
Pipeline creation
Deployment using Flask

A simple web interface is also integrated where users can enter student details and get predicted maths marks.

#### 🚀 Features

#### 📊 Workflow

1. Data Collection
    Dataset containing student-related information was used.
2. Data Preprocessing
    Handling categorical features
    Encoding
    Scaling
    Train-test split
3. Model Training
    Multiple regression models were trained and evaluated.
4. Model Evaluation
    Models were compared using evaluation metrics.
5. Pipeline Creation
    Complete training and prediction pipeline created.
6. Flask Integration
    Model deployed using Flask with a basic frontend.

#### 🌐 Web Application

A simple Flask web app is included where users can:

Enter student information
Submit data
Get predicted maths score instantly

Frontend is intentionally kept basic because the main focus of the project is on the Machine Learning pipeline and model building.
